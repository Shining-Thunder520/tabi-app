# イベタビ開発ドキュメント

## イベタビとは

イベタビは、推し活・舞台・ライブ・旅行などの遠征を管理するためのアプリです。

「次の遠征で本当に使いやすいこと」を最優先に、実際の遠征で感じた不便を改善しながら開発を進めています。

---

## コンセプト

- 推し活遠征をもっと快適にする
- 必要な情報を1つのアプリにまとめる
- 入力の手間をできるだけ減らす
- オフラインでも利用できる
- シンプルで分かりやすい操作

## 開発方針

- 実際の遠征で使いながら改善する
- 使いやすさを最優先にする
- 機能追加よりも操作性を重視する
- 仕様変更があった場合はドキュメントも更新する
